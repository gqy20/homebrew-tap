# homebrew-tap

`gqy20` 的 Homebrew tap，用于分发命令行工具。

## 安装

```bash
brew tap gqy20/tap
brew install zotcli
```

安装完成后可执行：

```bash
zot
```

## 可用 Formula

### `zotcli`

面向终端、脚本和 AI agent 的 Zotero CLI。

- Homebrew 包名：`zotcli`
- 安装后的命令：`zot`
- 项目仓库：`https://github.com/gqy20/zotero_cli`
